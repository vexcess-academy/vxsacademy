Var, Let, and Const

There are three keywords you can use to declare a variable in JavaScript: var, let, and const. A long time ago in the the primitive versions of JavaScript the only way to create a variable was using "var". However people didn't like the way that variables created with var were scoped. Scope is the area in the code where the variable exists. A variable does not exist outside of it's scope.

"var" is function scoped, while the newer "let" is block scoped. Try out the following piece of code. You will see that the program runs just fine and 123 is printed to the console. 

{
    var test = 123;
}
println(test);

Now replace "var" with "let". You will notice that the program now gives an error saying that "test is not defined" 

{
    let test = 123;
}
println(test);

The curly brackets aka braces {} define a block. A variable declared in the block using "var" will be useable outside of the block, but variables declared using "let" are only available inside of the block.

"const" is the same as "let" except that the variable you put into a "const" variable can NOT be changed. You can change variables created using "var" and "let", but if you try to put a new value into a constant variable it will throw an error.
