Data Types

The first thing we need to learn are the basic data types. These are the most primitive components that make up our programs. The primitive data types in JavaScript are the following:

    null - this value is a placeholder for an object undefined - this value is a placeholder for a variable that hasn't been assigned a value Boolean - this is value is either true or false Number - this is value stores a number String - this is value stores text BigInt - this is value can store massive integers that are too big to be stored in a regular Number Symbol - not even I know what this is; so don't worry about it

Out of those the ones you will use most often are Booleans, Numbers, and Strings. In addition to those primitive data types, there are also composite data types:

    Object - stores data in key/value pairs Array - a special type of object where each key is a number Function - a chunk of code that can be re-run many times

You most likely don't really understand what the explanations of each data type mean, but that's fine. It will make more sense when you see it in action and I go into more depth about each one in future lessons.
