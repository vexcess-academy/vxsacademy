Variable Naming Rules

Maybe you're wondering "can I name a variable whatever I want?". The answer to this is that you can name your variable ALMOST everything you want. Variable names can only include letters, numbers, underscores (_), and dollar signs ($). Also a fun fact is that you can use unicode characters such as Ï€ in your variable names. Also variable names can NOT start with a number. Also a variable can NOT be a reserved word such as "true", "false", "if", "for", "while", and many more. Because you can NOT use spaces ( ) in your variable names it is common practice to use underscores to seperate words in your variable name like so: example_variable_name. It is also common practice to use camelCase where you make the first letter of each word except for the first one capitalized like so: exampleVariableName. It is important to note that variable names are case sensitive. That means Hello and hello are two completely seperate variables.
